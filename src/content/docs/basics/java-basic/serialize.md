---
title: "Serialize"
---

Serialization is the process of converting an object into a transportable format. It is a means of data persistence. Generally widely used in network transmission, RMI and RPC scenarios.

Deserialization is the reverse operation of serialization.

Serialization is the process of converting object state information into a form that can be stored or transmitted. Generally transmitted in bytecode or XML format. Bytecode or XML encoding format can be restored to completely equal objects. This reverse process is called deserialization.
