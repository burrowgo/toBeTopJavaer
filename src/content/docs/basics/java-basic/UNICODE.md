---
title: "Unicode"
---

ASCII code only has 256 characters, which is fine for Americans as it includes almost all the characters they use. However, there are programmers other than Americans in the world, so a more comprehensive character set is needed.

Unicode (English: Universal Character Set) is an industry standard in the field of computer science. It organizes and encodes most of the world's writing systems, allowing computers to present and process text in a simpler way.

Unicode developed alongside the Universal Character Set standard and is also published in book form. Unicode is still being continuously revised, with each new version adding more characters. The latest version as of September 2023 is **15.1**, which includes over **149,000** characters. In addition to visual glyphs, encoding methods, and standard character encoding, Unicode data also includes character properties, such as uppercase and lowercase letters.

The development of Unicode is managed by the non-profit Unicode Consortium, which is dedicated to having Unicode schemes replace existing character encoding schemes. This is because existing schemes often have very limited space and are not suitable for multilingual environments.

Unicode is highly recognized and widely used in the internationalization and localization processes of computer software. Many new technologies, such as Extensible Markup Language (XML), the Java programming language, and modern operating systems, adopt Unicode encoding.

Unicode can represent Chinese characters.
