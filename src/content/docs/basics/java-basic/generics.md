---
title: "Generics"
---

Java generics are a new feature introduced in JDK 5 that allows the use of type parameters when defining classes and interfaces.

These declared type parameters are replaced with specific types when the class or interface is used. The primary application of generics is within the Java collections framework introduced in JDK 5.

The most significant benefit of generics is improved code reusability. For example, using the `List` interface, you can store `String`, `Integer`, or other types. Without generics, you would need a separate `List` interface for each data type. Generics solve this problem efficiently.
