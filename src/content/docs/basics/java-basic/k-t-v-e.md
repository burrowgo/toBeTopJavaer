---
title: "K T V E"
---

* **E** - Element (Used in collections because collections store elements)
* **T** - Type (Java class)
* **K** - Key (Key)
* **V** - Value (Value)
* **N** - Number (Numeric type)
* **?** - Represents an uncertain Java type (unbounded wildcard type)
* **S, U, V** - 2nd, 3rd, 4th types

**Object** is the root class of all classes. An object of any class can be assigned to an `Object` reference variable. When using it, type casting might be needed. However, after using generic identifiers like `T`, `E`, etc., the type is determined before actual use, so type casting is no longer necessary.
