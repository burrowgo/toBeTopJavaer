---
title: "Basic Data Types"
---

There are 8 basic data types in Java, divided into three major categories.

### Character Type

*   `char`

### Boolean Type

*   `boolean`

### Numeric Type

1.  **Integer Types**: `byte`, `short`, `int`, `long`
2.  **Floating-point Types**: `float`, `double`

*Note: `String` is not a basic data type; it is a reference type.*
