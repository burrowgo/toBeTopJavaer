---
title: "Keyword About Exception"
---

`throws`, `throw`, `try`, `catch`, `finally`

`try` is used to specify a block of code where exceptions are guarded against.

`catch` clauses immediately follow the `try` block and are used to specify the types of exceptions you want to catch.

`finally` ensures that a segment of code is executed regardless of whether an exception occurs.

The `throw` statement is used to explicitly throw an exception.

`throws` is used to declare the various exceptions that a method might throw.
