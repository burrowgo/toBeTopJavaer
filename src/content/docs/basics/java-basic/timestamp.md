---
title: "Timestamp"
---

A timestamp is a sequence of characters that can indicate that a piece of data existed before a certain point in time and is complete and verifiable. It uniquely identifies a specific moment in time.

A timestamp refers to the total number of seconds from Greenwich Mean Time 1970-01-01 00:00:00 (Beijing Time 1970-01-01 08:00:00) to the present. In common terms, a timestamp is a complete, verifiable set of data that can represent the existence of a piece of data at a specific point in time.
